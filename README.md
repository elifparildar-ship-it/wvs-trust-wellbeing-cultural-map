# Trust, Wellbeing, and Cultural Values (WVS Wave 7)

## Overview
This project presents a country-level analysis of the World Values Survey (Wave 7). 
It examines cross-national patterns in trust in government, life satisfaction, and broader cultural value dimensions using survey data and computational social science methods.

## Research Questions
- How does trust in government vary across countries?
- Is trust in government associated with life satisfaction at the country level?
- Can countries be grouped into meaningful cultural clusters based on shared values?

## Data
The analysis uses publicly available data from the World Values Survey (Wave 7).
The dataset includes individual-level survey responses aggregated to the country level for analysis.

Raw survey data are not included in this repository. Only aggregated results and analysis code are shared.

## Methods
- Data cleaning and aggregation to country-level indicators
- Interactive world maps (GIS) using `leaflet`
- Correlation and regression analysis
- Principal Component Analysis (PCA) to construct cultural value dimensions
- Network analysis and community detection to identify cultural similarity clusters
